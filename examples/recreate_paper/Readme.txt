Readme for how to recreate figures from the paper

run the following line:
python bin/generate_night_plan.py -d 2024-08-02 -f YOUR_PATH_HERE/optimalAllocation/examples/recreate_paper/real_2024B/ -ttp

The "-ttp" flag turns off the TTP computation. Since we are just interesting in solving the semester plan, no need to run the ttp here.
